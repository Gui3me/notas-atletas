# Notas dos Atletas

Projeto em JavaScript que calcula a média válida de atletas de ginástica artística.

## Regras do cálculo
-- Cada atleta recebe 5 notas
-- A maior e a menor nota são descartadas
-- A média é calculada com as 3 notas que sobraram

## Tecnologias
- JavaScript

## Como executar
Basta rodar o arquivo `notas-atletas.js` em um ambiente JavaScript ou no console do navegador.
